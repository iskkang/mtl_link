왜 MTL Link인가

MTL Link를 써야 하는 이유

기존 카카오톡·텔레그램·라인과 무엇이 다른지, 왜 우리 회사에 맞는지 알아보세요.

🔐

업무 데이터를 회사 서버에 보관

카카오톡·텔레그램 등 외부 서비스는 대화와 파일이 제3자 서버에 저장됩니다. MTL Link는 회사 전용 서버에서만 운영되어 정보 유출 위험이 없습니다.

🌏

자동 번역으로 언어 장벽 해소

한국어, 러시아어, 영어, 중국어, 우즈베크어 직원이 모두 한 앱에서 일합니다. 메시지를 보내면 상대방 언어로 자동 번역되어 전달됩니다.

🤖

물류 AI 어시스턴트 내장

MINT AI가 견적 누락 항목 체크, 운송 수단 추천, 통관 리스크 분석, 화물 추적 안내문 작성까지 도와줍니다. 따로 ChatGPT를 열 필요가 없습니다.

📁

업무 파일을 한 곳에서 관리

채팅방마다 파일이 정리되고, 파일 자료실에서 부서별·기간별로 검색할 수 있습니다. 첨부파일을 찾으러 카톡 스크롤을 내릴 필요가 없습니다.

📢

채널과 공지로 정보를 구조화

팀별 채널, 관리자 공지, 스레드 답글로 정보가 체계적으로 분류됩니다. 중요한 공지가 일반 메시지에 묻히지 않습니다.

✅

메시지가 업무로 연결

메시지를 할 일(액션 아이템)로 전환하고, 마감일을 설정하면 자동 리마인더가 옵니다. 대화에서 놓친 업무 지시가 없어집니다.

01  시작하기

처음 시작하는 방법

계정 발급부터 첫 메시지 전송까지, 5분이면 충분합니다.

1

계정 신청

MTL Link 접속 화면에서 "회원가입 신청"을 클릭하세요.
이름, 이메일, 부서, 직급을 입력하고 신청하면 관리자에게 알림이 전달됩니다.

ℹ️

관리자 승인 필수

보안을 위해 모든 계정은 관리자가 직접 승인합니다. 승인 완료 후 임시 비밀번호가 이메일로 발송됩니다.

2

임시 비밀번호로 첫 로그인

이메일로 받은 임시 비밀번호로 로그인하면 즉시 비밀번호 변경 화면이 나타납니다.
새 비밀번호로 변경하면 정상 이용이 시작됩니다.

  • 비밀번호: 8자 이상, 영문 대소문자 + 숫자 + 특수문자 포함
  • 이전 임시 비밀번호는 즉시 만료됩니다
3

프로필 설정

우측 상단 프로필 아이콘 → "프로필 편집"에서 사진과 아바타 색상을 설정하세요.
상태 메시지(예: "회의 중", "재택 근무")도 등록할 수 있습니다.

4

언어 설정

설정에서 나의 UI 언어와 수신 번역 언어를 선택하세요.
이 설정에 따라 상대방의 메시지가 내 언어로 자동 번역됩니다.

  • 지원 언어: 한국어 · English · Русский · 中文 · 日本語 · O'zbek
5

첫 메시지 보내기

왼쪽 멤버 목록에서 동료를 클릭하면 1:1 채팅방이 열립니다.
하단 입력창에 메시지를 입력하고 Enter를 누르면 전송됩니다.

📱

모바일에서도 사용 가능합니다 (PWA)

스마트폰 브라우저에서 MTL Link 주소를 열고 "홈 화면에 추가"를 선택하면 앱처럼 설치됩니다. 별도 앱스토어 설치 없이 네이티브 앱과 동일하게 사용할 수 있습니다.

02  주요 기능

이런 기능을 사용할 수 있어요

MTL Link의 모든 기능을 한눈에 확인하세요.

💬

1:1 · 그룹 채팅

멤버 목록에서 동료를 선택해 1:1 채팅을 시작하거나, 여러 명을 초대해 그룹방을 만들 수 있습니다. 메시지는 실시간으로 전달됩니다.

🌐

자동 번역

상대방 메시지 언어가 나의 설정 언어와 다르면 자동 번역됩니다. "원본 보기" 버튼으로 원문도 확인할 수 있습니다.

핵심 기능
🎤

음성 번역 메시지

마이크 버튼을 눌러 말하면 상대방 언어로 번역된 텍스트 메시지가 전송됩니다. 음성 파일은 서버에 저장되지 않습니다.

📎

파일 첨부

이미지, PDF, Excel, Word, PowerPoint, ZIP 파일을 채팅에 첨부할 수 있습니다. 드래그앤드롭도 지원합니다.

  • 이미지: 최대 10MB
  • 문서: 최대 30MB
  • 압축파일: 최대 50MB
🔗

링크 미리보기

URL을 입력하면 제목, 설명, 이미지가 포함된 미리보기 카드가 자동으로 표시됩니다. 선사 트래킹 링크, 뉴스 등에 유용합니다.

v2.24
📌

메시지 핀

중요한 메시지를 핀으로 고정하면 채팅방 헤더에서 언제든 빠르게 다시 찾을 수 있습니다.

🔁

스레드 & 답장

모든 답글은 스레드로 관리됩니다. 메시지 위에 마우스를 올리면 답장 버튼이 표시됩니다. 모바일에서는 길게 누르세요.

😊

이모지 반응

메시지에 이모지로 빠르게 반응할 수 있습니다. 팀원들의 반응을 한눈에 확인하세요.

🔔

알림 & DND

브라우저 푸시 알림과 소리 알림을 지원합니다. 방해금지(DND) 설정과 키워드 알림도 가능합니다.

✅

할 일 (액션 아이템)

메시지 컨텍스트 메뉴에서 "할 일로 등록"을 선택하면 마감일과 함께 태스크 목록에 추가됩니다. 24시간 미답변 시 자동 리마인더가 발송됩니다.

📢

채널 & 공지

팀별·업무별 채널을 통해 주제가 분리된 대화를 진행합니다. 관리자가 작성한 공지는 별도 공지 섹션에서 확인할 수 있습니다.

📁

파일 자료실

채팅에서 공유된 파일들이 파일 자료실에 자동으로 정리됩니다. 날짜, 파일 유형, 업로드한 사람별로 검색할 수 있습니다.

🔍

메시지 검색

채팅방 내 키워드 검색과 전체 채팅방에 걸친 글로벌 검색을 지원합니다. 파일명, 번역문까지 검색됩니다.

📅

캘린더 & 공휴일

6개국 (한국·미국·러시아·중국·일본·우즈베키스탄)의 공휴일이 캘린더에 동시 표시됩니다. 해외 거래처 스케줄 확인에 유용합니다.

💹

운임 대시보드

해운 운임 지수, 업계 뉴스, 세계 물동량, 항구 혼잡도를 대시보드 한 화면에서 확인합니다.

🌗

다크 / 라이트 모드

사이드바 하단 토글 버튼으로 다크모드와 라이트모드를 전환합니다. 눈의 피로도를 줄이는 데 도움이 됩니다.

03  MINT AI

AI 어시스턴트 MINT

물류 업무에 특화된 AI 도우미입니다. 사이드바의 MINT 아이콘을 눌러 시작하세요.

MINT — MTL AI 어시스턴트

견적 체크리스트, 운송 수단 추천, 통관 위험 분석, 화물 추적 안내문 작성까지. 물류 업무에 필요한 AI 기능을 모두 갖추고 있습니다. 대화 기록은 세션별로 독립 관리됩니다.

📋

견적 체크리스트

고객 문의 내용을 붙여넣으면 견적에 필요한 누락 정보를 자동으로 찾아줍니다.

✉️

메시지 작성

상황을 설명하면 거래처에 보낼 전문적인 물류 메시지 초안을 작성해 줍니다.

🚢

운송 수단 추천

화물 정보와 조건을 입력하면 해상·항공·육상 중 최적 운송 수단을 추천합니다.

🛃

통관 리스크 분석

품목과 목적지 국가를 알려주면 통관 위험도와 주의사항을 분석해 줍니다.

📦

화물 추적 안내문

화물 번호를 입력하면 운반사를 조회하고 고객에게 보낼 추적 안내문을 만들어 줍니다.

💬

자유 질의응답

물류 관련 질문이라면 무엇이든 물어보세요. 사내 지식 베이스를 참고해 답변합니다.

💡

MINT를 잘 활용하는 팁

질문이 구체적일수록 답변이 정확해집니다. "FCL 40ft 컨테이너, 상하이→부산, 기계류 10톤, 다음주 선적 예정"처럼 조건을 상세히 입력하면 더 좋은 결과를 얻을 수 있습니다.

04  활용 팁

더 효율적으로 쓰는 방법

알면 생산성이 달라지는 팁을 모았습니다.

⌨️

Enter vs Shift+Enter

Enter는 메시지 전송, Shift+Enter는 줄바꿈입니다.

📋

메시지를 할 일로 등록하기

메시지에 마우스를 올리거나 (모바일에서는 길게 누르면) 메뉴가 나타납니다. "할 일로 등록"을 선택하면 마감일과 함께 태스크 목록에 추가됩니다.

🔔

키워드 알림 설정하기

내 이름, 팀명, 특정 화물 번호 등 중요 키워드를 등록하면 해당 단어가 포함된 메시지가 오면 별도 알림을 받을 수 있습니다.

📎

파일 드래그앤드롭

파일 탐색기에서 파일을 채팅창으로 바로 드래그앤드롭하면 빠르게 전송됩니다. 버튼을 찾을 필요 없습니다.

🌐

번역 ON/OFF 개별 설정

채팅방 헤더의 언어 버튼을 눌러 방마다 번역 언어를 다르게 설정할 수 있습니다. "번역 안 함"도 선택 가능합니다.

🔍

전체 메시지 검색 활용하기

사이드바 검색 아이콘을 클릭하면 모든 채팅방을 한 번에 검색합니다. 오래된 첨부파일이나 공유된 링크를 찾을 때 유용합니다.

📌

중요 메시지는 핀으로 고정

배송 일정, 계좌번호, 중요 안내 등은 핀으로 고정해 두세요. 채팅 헤더의 핀 아이콘을 클릭하면 고정된 메시지 목록이 표시됩니다.

💬

채널 요약으로 빠르게 파악하기

채널에서 MINT에게 "오늘 대화 요약해줘"라고 하면 채널 내용을 정리해 줍니다. 자리를 비운 후 빠르게 캐치업하는 데 유용합니다.

05  자주 묻는 질문

FAQ

사용하다 막히면 먼저 여기를 확인하세요.

비밀번호를 잊어버렸어요. ▾
관리자에게 비밀번호 재설정을 요청하세요. 관리자가 재설정하면 새 임시 비밀번호가 이메일로 발송됩니다.
메시지 번역이 안 되거나 이상해요. ▾
설정에서 나의 수신 언어가 올바르게 설정되어 있는지 확인하세요. 채팅방 헤더의 언어 설정도 확인하세요. 특정 번역이 부자연스러울 때는 원본 보기 버튼으로 원문을 확인하고 관리자에게 피드백을 주시면 개선에 반영됩니다.
알림이 오지 않아요. ▾
브라우저 알림 권한이 허용되어 있는지 확인하세요. 브라우저 주소창 옆 자물쇠(🔒) 아이콘을 클릭하면 알림 권한을 변경할 수 있습니다. 또한 앱 설정에서 DND(방해금지) 모드가 켜져 있지 않은지도 확인하세요.
파일 업로드가 안 돼요. ▾
파일 크기 제한 (이미지 10MB / 문서 30MB / 압축파일 50MB)을 초과했거나, 허용되지 않은 파일 형식(exe, bat, js 등)일 수 있습니다. 파일 크기를 줄이거나 PDF로 변환 후 시도해 보세요.
음성 메시지 번역이 제대로 안 돼요. ▾
마이크 권한이 브라우저에 허용되어 있는지 확인하세요. 배경 소음이 없는 환경에서 또렷하게 말하면 인식률이 높아집니다. 최대 녹음 시간은 60초입니다.
채팅방을 실수로 나갔어요. ▾
1:1 채팅방은 다시 멤버 목록에서 해당 직원을 클릭하면 기존 대화가 이어집니다. 그룹방의 경우 방을 나가면 재초대가 필요할 수 있으니 방 관리자에게 요청하세요.
MINT AI가 이상한 답변을 해요. ▾
MINT는 일반적인 물류 지식을 기반으로 답변하며, 회사 내부 데이터(운임, 계약, 고객 정보)는 알지 못합니다. 구체적인 수치나 계약 조건이 필요한 경우에는 참고용으로만 활용하고 내부 데이터를 우선하세요.
민감한 업무 내용을 MINT나 번역 기능에 입력해도 되나요? ▾
번역과 AI 기능은 외부 API(Claude, Whisper)를 경유합니다. 고객 개인정보, 계약 조건, 금융 정보 등 매우 민감한 내용은 AI 기능 사용을 자제하세요. 일반적인 업무 대화와 물류 문의는 정상적으로 사용하셔도 됩니다.
모바일에서 앱처럼 설치할 수 있나요? ▾
네, 가능합니다. 스마트폰 브라우저에서 MTL Link 주소를 열고 브라우저 메뉴에서 "홈 화면에 추가" 또는 "앱으로 설치"를 선택하면 홈 화면에 아이콘이 생기고 앱처럼 실행됩니다. iOS Safari와 Android Chrome 모두 지원합니다.
