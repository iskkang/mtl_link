# POA (Power of Attorney / 위임장) 가이드

## 적용 범위

CIS 지역 운송 시 수입자(Consignee)가 현지 통관 대리인에게 권한을 위임하는 서류.
러시아, 카자흐스탄, 우즈베키스탄 등 CIS 통관 필수 서류.

---

## POA 기본 요건

- [ ] **러시아어로 작성** (영문 병기 가능하나 러시아어 원문 필수)
- [ ] 위임인(Consignee) 정보 기재
  - 회사명 (러시아어 표기)
  - 납세자식별번호 (ИНН)
  - 회사 주소
  - 사업자등록번호 (ОГРН)
- [ ] 수임인(통관 대리인) 정보 기재
- [ ] 위임 권한 범위 명시
- [ ] 유효기간 명시
- [ ] 대표자 서명 + 회사 직인
- [ ] (요청 시) 공증(Notarization)

---

## 위임 권한 범위 (표준)

러시아어 POA에 포함되어야 할 주요 권한:
- 수출입 서류 수령 (BL, 통관 서류, 선하증권 등)
- 화물 조작, 보관, 인도 관련 서류 수령
- 결제 서류 수령
- 화물 조작 및 운송 주선
- 세관 제시 및 통관 검사 참석
- 세관 이의 신청 권한

---

## 지역별 특이사항

### 카자흐스탄
- POA **공증본(Notarized)** 필수
- 일반 서명본 불가 → 출발 전 공증 완료본 반드시 수령
- 부킹 후 3일 내 공증본 수령 확인 필수

### 러시아
- 공증 필수 여부는 통관 에이전트에 사전 확인
- 일부 케이스는 일반 직인본으로 가능

### 우즈베키스탄
- EAC 인증 등 별도 수입 라이선스와 함께 제출 필요한 경우 있음

---

## POA 체크리스트 (MINT 운영)

```
CIS 행 화물 처리 시 반드시 확인:

[ ] POA 수령 여부
[ ] 러시아어 작성 여부
[ ] 공증 여부 (Kazakhstan: 필수, Russia: 케이스별)
[ ] 유효기간 확인
[ ] 수임인 정보 일치 여부 (통관 에이전트와 동일한지)
```

---

## 자주 발생하는 문제

| 문제 | 결과 | 예방 |
|---|---|---|
| POA 공증 미비 (KZ) | 출발 보류 | 수주 후 즉시 공증 요청 |
| POA 유효기간 만료 | 통관 보류 | 출발 전 유효기간 확인 |
| 수임인 정보 불일치 | 통관 거부 | 통관 에이전트 정보 확인 후 POA 작성 |
| 러시아어 미작성 | 통관 거부 | 러시아어 버전 의무 요청 |
