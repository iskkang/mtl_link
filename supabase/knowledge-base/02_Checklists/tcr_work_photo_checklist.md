# TCR 컨테이너 작업사진 체크리스트

## 적용 범위
TCR(Trans China Railway) 운송 진행 시 화주/CFS에 요청하는 작업사진 기준.
중국 세관 제출용이며 **미충족 시 한국으로 반송될 수 있음.**

## 필수 사진 목록 (최소 8장)

- [ ] **1. 공컨테이너 내부** — 컨테이너 번호 보이게, Door Open 상태
- [ ] **2. 절반 적입 (1/2)** — 컨테이너 번호 보이게
- [ ] **3. FULL 적입 완료** — 컨테이너 번호 보이게
- [ ] **4. 쇼링 및 고박 완료** — 컨테이너 번호 보이게, 단단하게 고정된 상태
- [ ] **5. 한쪽 문 닫은 사진** — 컨테이너 번호 전체적으로 보이게 (필수)
- [ ] **6. 양쪽 문 닫은 사진** — SEAL 봉인 완료 상태
- [ ] **7. SEAL 사진** — 씰 번호 식별 가능하게
- [ ] **8. CSC SAFETY APPROVAL 사진** — 컨테이너 CSC 플레이트

## 핵심 주의사항

```
★ 모든 사진(CSC 제외)에는 반드시 컨테이너 번호가 보이게 촬영
★ SEAL은 반드시 말뚝씰(Bolt Seal) 사용
★ SEAL 부착 위치: 오른쪽 문(Right Door), 왼쪽 레버(Left Lever)
★ Cargo & PLT Size 명세 함께 제공 필수 (장입 구조도 작성용)
```

## TSR과의 차이점

| 항목 | TCR | TSR |
|---|---|---|
| 최소 사진 수 | 8장 | 7장 |
| CSC 사진 | **필수** | 불필요 |
| SEAL 종류 | 말뚝씰 | 말뚝씰 |
| 제출 대상 | 중국 세관 | 철도청 |

## 사진 미제출 결과

- 중국 세관 측 미충족 시 → **한국 반송 가능**
- 철도 INSPECTION 지정 → 추가 비용 + 발차 지연
