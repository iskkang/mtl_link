# TCR 서류 & 중량 체크리스트

## 적용 범위
TCR 운송 진행 시 출발 전 서류 및 중량 확인.
MTL 공식 화주 안내문 기준.

---

## Part 1: 선적 서류 필수 기재사항 (CI/PL)

- [ ] **HS CODE** — 수출국 기준 + 수입국 기준 각각 기재
- [ ] **Item별** Net Weight, Gross Weight, Invoice Value 분할 기재
- [ ] Container No. & Seal No. & CBM 기재
- [ ] **SEAL 장착 위치** 확인
- [ ] CONSIGNEE: 상호명, 주소, 전화번호 상세 기재
- [ ] FINAL DESTINATION & STATION CODE 기재

> 여러 컨테이너를 한 건의 서류로 작성할 경우:
> **컨테이너당 Net Weight, Gross Weight, Invoice Value 반드시 분할**
> 총 중량, 총 수량, CBM도 기재

---

## Part 2: 철도 운송 필수 서류

컨테이너에 부착 후 운송 → 도착지 통관용

- [ ] **Commercial Invoice** (영문 + 러시아어 + HS CODE)
- [ ] **Packing List** (영문 + 러시아어 + HS CODE)
- [ ] **Railway Bill** (컨테이너당 1부 발행)
- [ ] Certificate / C/O — 품목에 따라 필요 여부 확인

---

## Part 3: TCR 중량 제한 (필수 확인)

| 컨테이너 | 최대 중량 |
|---|---|
| 20'FT | **21.5 TON** |
| 40'FT | **25 TON** |

### 편중/편재 제한
- [ ] 전후방 무게 차이: **5톤 초과 불허** (편중)
- [ ] 화물 무게중심: 좌측 또는 우측 **90mm 초과 불허** (편재)
- [ ] 20'FT × 2대 작업 시: 두 컨테이너 무게 차이 **3.5톤 초과 불허**

> ⚠️ 편중/편재 발생 시 → 컨테이너 INSPECTION + 재작업
> → **발차 지연 + 추가 비용 발생**

---

## Part 4: 노선별 중량 추가 확인

엑셀 매뉴얼 기준 (운임 협의 시 별도 확인):

| 서비스 | VGM 기준 | 최대 | 특이사항 |
|---|---|---|---|
| TCR ALL RAIL | 30톤 미만 | 30톤 | 가능하면 29톤 미만 권장 |
| TCR RAIL+TRUCK | 23톤 기준 | 30톤 | 초과 시 사전 확인 필수 |
| TCR ALL TRUCK | 23톤 기준 | - | 작업사진 촬영 권장 |

- [ ] 패킹당 중량 1,000kg 이상 → 철도역 체크 필요
- [ ] 패킹당 중량 5톤 미만 선적 가능 (단, 사전 컨펌 필수)
- [ ] **중량화물 포장 타입 사전 확인 및 컨펌 필수**
