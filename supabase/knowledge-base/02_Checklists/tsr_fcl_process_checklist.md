# TSR FCL 업무 진행 체크리스트

## 적용 범위

TSR FCL 운송 진행 시 단계별 확인 항목.
NOMI 건(노미네이션)과 LOCAL 건으로 구분.

---

## Phase 1: 선적 준비

- [ ] **NOMI 건**: 파트너로부터 Booking Sheet 수령 (반드시 먼저 받은 후 진행)
- [ ] SHPR 컨택 → Cargo Readiness Date + 화물 디테일 수령
- [ ] 선적/진행 스케줄 확인 (Via Yingkou or Via Changchun)
- [ ] Container Loading Plan에서 사용할 EMPTY CNTR No., Release No. 확인
- [ ] Container Loading Plan 공란에 체크

---

## Phase 2: 부킹 및 배차

- [ ] 배차 완료 (FCL: ING/극동티엘에스 / LCL: ING/하나파인물류)
- [ ] 선사 부킹 완료 (범영훼리 등 / S/R, VGM 전달 필요)
- [ ] 반입 확인

---

## Phase 3: 서류 마감

- [ ] 화주로부터 선적서류 (CI/PL) 수령
- [ ] 수출통관서류 진행/인수 완료
- [ ] H B/L, M B/L, S/R, VGM 입력 완료
- [ ] 파트너측에 Cargo Readiness Date, ETD, ETA, Schedule 전달
- [ ] Booking Form 작성 + 공용폴더 파일 정리
- [ ] EDI 신고 완료 (마감 시간 확인 필수)

---

## Phase 4: 선적 확인

- [ ] CHECK HOUSE B/L 화주 확인 완료
- [ ] CHECK MASTER B/L 파트너 확인 완료
- [ ] FELB에 부킹 완료 (CI/PL, M B/L, H B/L, Booking Form)
- [ ] Final B/L + 수출면장 화주 전달
- [ ] 파트너 PRE-ALERT 발송 (FELB 노미 건은 불필요)

---

## Phase 5: 정산

- [ ] 화주에게 운송 관련 비용 청구
- [ ] 세금계산서 발행
- [ ] 파트너에게 DN (Debit Note) 전달
- [ ] DN 컨펌 수령

---

## Phase 6: 트레이싱

- [ ] Tracing File (FCL, EX-KOREA, DIMENSION TAB) 작성
- [ ] Tracing File 고객사 교부
- [ ] YL 물류에서 LCL CLP Stuffing 사진 + CLP 파일 수령 (LCL 건)
- [ ] CLP File 작성 완료 (LCL 건)

---

## MINT 체크 포인트

운영팀이 TSR FCL 진행 건을 MINT에 입력하면:
```
1. Phase 1~6 중 어느 단계인지 확인
2. 현재 단계의 체크 항목 중 누락된 것 확인
3. 다음 단계 액션 안내
```
