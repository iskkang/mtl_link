# ATA Carnet (ATA까르네) 가이드

## 정의

ATA Carnet(ATA까르네)는 **물품의 무관세 임시통관 증서**입니다.

ATA 협약 가입국 간에 물품을 일시적으로 수입/수출/보세운송하기 위해
복잡한 통관 서류나 담보금 없이 통관을 신속하게 처리하는 국제 제도.

---

## 용도 (3가지만 가능)

| 용도 | 설명 |
|---|---|
| **상품견본** (Commercial Samples) | 판매 목적 없는 샘플 |
| **직업용구** (Professional Equipment) | 기자재, 촬영장비, 전시용 기계 등 |
| **전시회** (Fairs/Exhibitions) | 박람회, 전시회 출품물 |

> ❌ 사용 불가: 농산물, 식료품, 위험물품, 소모품, 1회용품, 수입 금지 품목

---

## 유효기간

- 발급일로부터 **최장 1년** (연장 불가)
- 수입국 세관이 재수출 기간을 명시하면 그 기간 내 반드시 재수출
- 중국의 경우: 보통 3개월, 최장 6개월

|---|---|
| CIS | Russia(RU), Ukraine(UK) |
| 유럽 | France(FR), Germany(DE), Poland(PL), UK(GB), Italy(IT), Spain(ES) 외 다수 |

---

## 운송 서류 구성

ATA Carnet는 다음 부분으로 구성:

| 구분 | 영문 | 내용 |
|---|---|---|
| 표지 | Cover | 발급 정보, 유효기간, 사용 가능 국가 |
| 수출 용증 | Exportation Voucher | 한국 세관 수출 신고 |
| 수입 용증 | Importation Voucher | 방문국 세관 수입 신고 |
| 재수출 용증 | Re-exportation Voucher | 방문국 세관 재수출 신고 |
| 재수입 용증 | Re-importation Voucher | 한국 세관 재수입 신고 |
| 총괄목록 | General List | 품목 목록 (품명, 수량, 가액, 원산지) |

## CI/PL 기재 항목 (General List 기준)

- [ ] Item 번호
- [ ] 품명, 규격 및 고유번호 (시리얼번호 등)
- [ ] 수량
- [ ] 중량 또는 부피
- [ ] 가액 (상업적 가치)
- [ ] 원산지 (ISO 국가코드)

---

## 운영팀 체크리스트

### 수출 전

- [ ] ATA Carnet 적용 대상 용도인지 확인 (샘플/직업용구/전시품)
- [ ] 목적국이 ATA 협약 가입국인지 확인
- [ ] 대한상공회의소에 발급 신청
- [ ] 유효기간 내 귀국 일정 확인

### 수출 시

- [ ] 한국 세관에서 수출 용증 처리
- [ ] Carnet를 화물과 함께 지참

### 목적국 입국 시

- [ ] 해당국 세관에서 수입 용증 처리
- [ ] 재수출 기한 명시 확인

### 귀국 시

- [ ] 목적국 세관에서 재수출 용증 처리
- [ ] 한국 세관에서 재수입 용증 처리
- [ ] **사용 후 반드시 대한상공회의소에 반납**

---

## MINT 판단 규칙

ATA Carnet 관련 질문이 오면:

```
Quick 모드 응답:
1. 용도 확인 (샘플/직업용구/전시품 해당 여부)
2. 목적국 협약 가입 여부 안내
3. 유효기간 1년 안내
4. 발급 기관 안내 (대한상공회의소)

MINT가 하지 않는 것:
- 발급 가능 여부 확정 (관세사/대한상공회의소 확인 필요)
- 세관 처리 결과 예측
```
