# 중고차 TCR 특수라싱 체크리스트

## 적용 범위
중고차(Used Car) TCR 운송 시 컨테이너 작업 기준.
**⚠️ 오일 누유 시 $5,000 이상 페널티 + 발차 불가 (백쉽 조치)**

---

## Part 1: 작업 전 필수 확인

- [ ] **컨테이너 바닥 모포 작업 필수** (중고차 운송 시 항상)
- [ ] 단중 2.5톤 이상 화물 → **사전 MTL 통보 필수**
- [ ] VGM **26TON 초과 불허** (초과 시 발차 불가)
- [ ] 컨테이너 전후 무게 차이 **3톤 미만**
- [ ] 좌우 기울기 **40mm 이내**

---

## Part 2: 배터리 처리 (필수)

차량에 아래 배터리 종류가 장착된 경우:
- 차량 배터리 연결선 **해체 후** 전선 부분에 **테이핑** 처리
- 테이핑된 사진 제공 필수

해당 배터리 종류: 자동차용 일반 납산 배터리 (밧데리)

---

## Part 3: 작업사진 (10장, 철도청 요청)

- [ ] **1.** EMPTY 컨테이너 내부 (컨테이너 번호 + 상태 식별 가능)
- [ ] **2.** 첫 번째 줄 작업 완료
- [ ] **3.** 1/4 화물 적입
- [ ] **4.** 2/4 화물 적입
- [ ] **5.** 3/4 화물 적입
- [ ] **6.** 화물 적입 완료 후 양쪽 문 모두 열린 상태
- [ ] **7.** 오른쪽 문만 닫고 왼쪽 열린 상태 (화물+컨번호 동시 촬영) — **★중요★**
- [ ] **8.** 막대씰 채우고 전체 컨테이너 사진
- [ ] **9.** 막대씰 채우고 클로즈업 사진
- [ ] **10.** CSC PLATE 사진

---

## Part 4: 적입 기준

- [ ] 적입사진 각각에 **화물 라싱된 모습** 보여야 함
- [ ] 1층 적입화물 간 공간 남을 시 → **화물 하단에 삼각목** 받치기
- [ ] 2층 화물도 **라싱 필수**

---

## 위반 시 결과

| 상황 | 결과 |
|---|---|
| 오일 누유 발생 | $5,000 이상 페널티 + 발차 불가 + 백쉽 조치 |
| 26TON 초과 | 발차 불가 |
| 편중/편재 초과 | INSPECTION + 재작업 + 발차 지연 |
