# TSR CIS 체크리스트 (서류 + 작업사진)

## 적용 범위
TSR(Trans-Siberian Railway) CIS 향 운송.
Busan → Vladivostok → Tashkent/Almaty/기타 CIS 최종 목적지.

---

## Part 1: 서류 체크리스트

### CI/PL 필수 기재사항
- [ ] 영문 + **러시아어** 동시 작성 (러시아어 버전 필수)
- [ ] 최종 CI/PL 상에 **SHIPPER 서명** 필수
- [ ] 컨테이너별로 CI/PL 각각 작성 (컨 통합 불가)
- [ ] **HS CODE 별로** Net Weight / Gross Weight / Value of Goods 분리 기재
- [ ] 실제 화물 Gross Weight와 서류 상 차이 없도록 작성
- [ ] CONSIGNEE 상호명, 주소, 전화번호 상세 기재
- [ ] FINAL DESTINATION & STATION CODE 기재

> ⚠️ 서류와 실제 계근값 오차 크면 → FULL INSPECTION 지정 →
> 컨테이너 전체 화물 적출 후 개별 중량 측정 (비용 발생)

---

## Part 2: 컨테이너 작업 기준

### 중량 제한
- [ ] One Package **1.5톤 이내** 작업
- [ ] 편중 **800kg 미만** (좌우 무게 차이)
- [ ] 1.5톤 초과 화물 → **TSR 특수라싱 필수**

### 작업 기준
- [ ] 철송 기준 쇼링 작업 완료 (철송은 흔들림이 심함)
- [ ] 화물 간 고박 완료

---

## Part 3: 작업사진 체크리스트 (최소 7장)

- [ ] **1. 공컨테이너** (Door Open, 컨테이너 번호 보이게)
- [ ] **2. 스터핑 작업 중간 사진** (작업 진행 중 여러 장)
- [ ] **3. 쇼링 전 사진** (컨테이너 번호 보이게)
- [ ] **4. 쇼링 완료 양문 개방** (컨테이너 번호 보이게)
- [ ] **5. 한쪽 문 닫힌 사진** (컨테이너 번호 전체 보이게 — 필수)
- [ ] **6. 양문 닫힌 사진** (SEAL 봉인 완료)
- [ ] **7. SEAL 사진** (씰 번호 식별 가능)

```
★ SEAL 부착 위치: 컨테이너 우측 DOOR의 좌측 LEVER
★ HIGH SECURITY SEAL (BARREL SEAL) 사용
★ SEAL 사진 제외하고 모든 사진에 컨테이너 번호 보이게 촬영
```
