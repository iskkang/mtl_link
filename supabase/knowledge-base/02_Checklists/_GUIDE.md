# 02_Checklists 작성 가이드

## Phase 1 작성 대상 (5개)

| 파일명 | 우선순위 | 비고 |
|---|---|---|
| `invoice_checklist.md` ✅ | High | 완료 (참고용) |
| `packing_list_checklist.md` | High | Invoice와 cross-check |
| `bl_checklist.md` | High | 운송 서류 핵심 |
| `cmr_checklist.md` | Medium | 도로 운송 |
| `cis_customs_checklist.md` | High | CIS 지역 특이 |
| `dg_cargo_checklist.md` | Medium | 위험물 |
| `used_car_checklist.md` | Low | 특수 화물 |

## 작성 원칙

1. **`- [ ]` 체크박스 형식** 사용
2. 섹션을 명확히 구분 (Header / Cargo Info / Trade Terms / 등)
3. **일치성 확인 섹션** 필수 (다른 서류와 cross-check)
4. **지역별 추가 요구** 섹션 (해당 시)
5. **자주 발생하는 실수** 섹션 — 운영 노하우 누적
6. 메타데이터 frontmatter 필수
