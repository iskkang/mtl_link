# SEAL 위치 및 관리 규정

## 목적

러시아/CIS 향 화물 SEAL 오부착 시 러시아 PORT에서 세관 검사 및 부대비용이 발생합니다.
본 규정은 SEAL 관련 오류를 사전에 방지하기 위한 기준입니다.

---

## SEAL 종류

**러시아 향 화물에는 반드시 High Security Seal (막대실) 사용**

| 구분 | 설명 |
|---|---|
| High Security Seal (막대실) | 러시아 향 필수. 금속 막대 형태의 고보안 씰 |
| 말뚝씰 (Bolt Seal) | TCR 일반 사용 |

---

## SEAL 부착 위치 (절대 규칙)

```
위치: 컨테이너 오른쪽 Door의 왼쪽 Locking Bar
      (Right Door, Left Locking Bar)
```

- Door의 Locking Bar 4개를 모두 올바르게 닫기
- Lock Door Cam 부분 확실히 잠그기

---

## SEAL 번호 통보 기준

- SEAL No. = **일련번호 + 앞에 위치한 영문자 포함**
- 예: `T 800003` (T와 800003 모두 포함)
- 같은 줄에 있는 영문+번호가 최종 SEAL No.
- 윗줄에 별도로 적힌 영문은 포함하지 않음

---

## 컨테이너 DOOR 관리 규정

부킹시트 전달 시 함께 안내:
- DOOR Locking Bar 4개 모두 올바르게 잠금
- Lock Door Cam 확실히 체결
- 미준수 시 러시아에서 추가 비용 청구 가능

---

## 위반 시 결과

| 상황 | 결과 |
|---|---|
| SEAL 정위치 미부착 | 러시아 PORT 세관 검사 + 부대비용 |
| SEAL No. 오류 (영문자 미포함 등) | 러시아 PORT 세관 검사 + 부대비용 |
| RE-SEALING 필요 | 운송 지연 + 추가 비용 발생 |
| DOOR 잠금 불량 | 추가 비용 책임 PARTY 청구 가능 |

---

## 운영팀 체크리스트

출발 전:
- [ ] SEAL 종류 확인 (러시아/CIS 향: High Security Seal)
- [ ] SEAL 부착 위치: 오른쪽 Door, 왼쪽 Locking Bar
- [ ] SEAL No. 정확히 기록 (영문 포함)
- [ ] 작업사진에 SEAL 봉인 사진 포함
- [ ] 화주/운송사에 SEAL 위치 공문 전달 여부 확인

서류 제출 시:
- [ ] BL/SR에 기재된 SEAL No.와 실제 SEAL No. 일치 확인

---

## MINT 판단 규칙

```
러시아/CIS 향 화물에서 다음 상황 발생 시:
- SEAL 관련 문의 → 본 규정 안내
- RE-SEALING 통보 → Medium risk
  → 화주에게 추가 비용 가능성 안내
  → 발차 지연 가능성 안내
- SEAL 오류로 세관 검사 지정 → High risk
  → 팀장 보고
```
