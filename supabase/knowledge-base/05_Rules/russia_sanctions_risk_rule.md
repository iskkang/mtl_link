# 러시아/벨라루스 제재 리스크 규칙

## 목적

對 러시아·벨라루스 제재는 수시로 강화되고 있습니다.
MINT는 제재 관련 법률 판단을 할 수 없으나, 리스크 신호를 감지하고 에스컬레이션합니다.

---

## 1. 제재 관련 주요 현황 (참고)

- **2024.2.23** 미국 재무부/국무부 고강도 對러시아 제재 발표
- **VSC** (Vostochnaya Stevedoring Company) — 제재 지정
- **TC** (Trans-Container) — 제재 지정 (러시아 최대 철도 운송사)
- EU 13차 對러시아 제재 발표

> ⚠️ 제재 목록은 수시 변동. 항상 최신 정부 고시 확인 필요.

---

## 2. MINT 판단 규칙

### 자동 High/Critical 분류 상황

| 상황 | Risk Level | 조치 |
|---|---|---|
| 화물이 러시아/벨라루스 행인데 HS코드 확인 안 됨 | High | 상황허가 대상품목 확인 요청 |
| 제재 대상 품목 의심 (전자부품, 반도체, 이중용도 등) | Critical | 즉시 팀장/대표님 보고 |
| VSC 또는 TC 경유 운송 계획 | High | 대체 파트너 확인 요청 |
| 화주가 최종 목적지 불명확 | High | 최종 수하인 확인 요청 |
| 러시아 행인데 수입자 정보 불충분 | High | 수입자 정보 보완 요청 |

### MINT가 하지 않는 것

- ❌ "이 화물은 제재 대상입니다" — 법률 판단 금지
- ❌ "이 HS코드는 반출 가능합니다" — 확인 권한 없음
- ✅ "이 화물은 상황허가 대상품목 여부 확인이 필요합니다" — 확인 요청

---

## 3. 상황허가 대상품목 확인 방법

**MINT가 직접 판단하지 않고 반드시 담당자가 확인:**

1. 산업통상자원부 전략물자관리시스템 확인
   - 웹사이트: yestrade.go.kr
2. 對러시아·벨라루스 상황허가 대상품목 HS Code 연계표 확인
   - 버전: 2024.02 기준 (최신 버전 항상 확인)
3. 불확실한 경우 → 전략물자관리원 문의 또는 관세사 자문

---

## 4. TCR 대안 활용

TSR 대신 TCR을 대안으로 적극 활용 (제재 영향 최소화):

```
TSR 리스크:
- VSC (블라디보스톡 터미널) 제재 지정
- TC (Trans-Container) 제재 지정
- 블라디보스톡 혼잡 증가

TCR 대안:
- 중국 경유 루트 (Qingdao/Lianyungang → Khorgos)
- 제재 직접 영향 낮음
- 다만 TCR 적체 증가 가능 → 리드타임 여유 확인
```

---

## 5. 화주 안내 규칙

러시아/벨라루스 행 화물에 대해 화주에게 반드시 안내:

```
안내 필수 사항:
1. 상황허가 대상 여부 화주 자체 확인 요청
2. 최종 수하인 정보 상세 제공 요청
3. 운송 중 추가 제재 시 중단 가능성 사전 고지
4. 확정 표현 금지 ("반드시 도착합니다" 등)
```

---

## 6. 에스컬레이션

제재 관련 의심 상황은 **무조건 즉시 팀장/대표님 보고**:
- 화물이 제재 대상품목 가능성 있는 경우
- 화주가 최종 목적지/수하인 공개 거부
- 통관에서 제재 관련 보류 발생

---

## 자기 점검 (러시아/벨라루스 행 모든 케이스)

```
[ ] HS코드 상황허가 대상 여부 확인됐는가?
[ ] 최종 수하인 정보 상세 확인됐는가?
[ ] VSC/TC 경유 여부 확인됐는가?
[ ] 화주에게 제재 리스크 사전 안내했는가?
[ ] 불확실 사항이 하나라도 있으면 에스컬레이션했는가?
```
