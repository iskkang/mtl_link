# 철도 운송 OBL 특수 규칙

## 핵심

러시아/CIS 철도 운송은 BL 없이 수하인에게 화물이 자동 인도됩니다.
해상 운송과 완전히 다른 특성입니다.

## 철도 운송 BL 특성 (러시아·CIS 철도법 기준)

철도차량에 화물 상차 후 발차 → 수취권이 수하인에게 자동 이전

즉:
- BL 인도/제시 없이 화물이 수하인에게 인도됨
- BL 배서 없이도 BL에 기재된 수하인이 화물을 수취할 권리를 가짐

## 화주 사전 안내 필수 사항

1. 화물 홀딩이 필요하면 발차 전 MTL과 반드시 사전 확인
2. 홀딩 시 보관료 포함 모든 추가 비용은 화주 부담
3. OBL HOLDER로서 "OBL 반납 없이 수하인에게 인도되어도 소유권 주장 안 함" 확인 서명 요청

## OBL 공문 기재 항목

- BL No. / Vessel/Voyage / SHPR / CNEE / 실제 선적일 / 요청일자 / 회사 명판/직인

## MINT 판단 규칙

CIS 철도 운송 화주 문의 시:
- BL 없이 화물 수취 문의 → OBL 특성 안내
- 홀딩 요청 → "발차 전 MTL 확인 필수, 추가 비용 화주 부담" 안내
- OBL 관련 분쟁 가능성 → High risk, 매니저 보고
