# Korea → CIS (TSR 노선 가이드)

## 1. 노선 기본 정보

```
POL: Incheon / Busan
POD (RU): Vladivostok (블라디보스톡)
Final: Tashkent / Almaty / 기타 CIS
운송 방식: 해상 + TSR 철송
```

---

## 2. 제재 리스크 주의 (2024년 기준)

⚠️ **VSC(Vostochnaya Stevedoring Company)** — 미국 제재 지정
⚠️ **TC(Trans-Container)** — 미국 제재 지정 (러시아 최대 철도 운송사)

→ VSC/TC 경유 사용 시 사전 대표님 확인 필수
→ **TCR을 대안 루트로 적극 검토**

---

## 3. 진행 프로세스 (NOMI 건)

```
1. 현지 지사로부터 부킹 요청
   ↓
2. 수출자에게 출하일정, 화물 디테일, 운임 조건 확인
   ↓
3. 선사 스케줄 + TSR 진행 가이드 안내
   ↓
4. 선사 부킹 (★SOC 부킹 필수★)
   ↓
5. CI/PL/면장 수령 → 선사 S/R 마감
   ↓
6. HBL 전달 + 확인 요청
   → 에이전트에 MBL/HBL/CI/PL + Empty Return Instruction + 작업사진 전달
   ↓
7. 선적일: MBL SURRENDER 또는 SEA WAYBILL 처리
```

---

## 4. CI/PL 작성 기준

- [ ] 영문 + **러시아어** 동시 작성
- [ ] 최종 CI/PL 상 **SHIPPER 서명** 필수
- [ ] **컨테이너별로** CI/PL 각각 작성
- [ ] **HS CODE별로** Net/Gross Weight, Value of Goods 분리
- [ ] 실제 Gross Weight와 서류 차이 없게 작성

---

## 5. 컨테이너 작업 기준

| 항목 | 기준 |
|---|---|
| 패키지당 최대 중량 | 1.5톤 이내 |
| 편중 | 800kg 미만 |
| 1.5톤 초과 화물 | TSR 특수라싱 필수 |
| 쇼링 | 철송 기준 필수 (흔들림 심함) |

## 6. 필수 제출 서류

| 서류 | 필수 여부 | 비고 |
|---|---|---|
| MBL / HBL | ✅ | |
| CI/PL | ✅ | 영문 + 러시아어 |
| 작업사진 | ✅ | 최소 7장 |
| Certificate / MSDS | 조건부 | 해당 품목 |
