# Korea → Uzbekistan 노선 가이드

## 1. 노선 기본 정보

| 루트 | 경유 | 리드타임 | 특징 |
|---|---|---|---|
| **TCR** | Busan → LYG/Qingdao → Tashkent | 30~35일 | 40피트 기준, 20피트 pair만 가능 |
| **TSR** | Busan → Vladivostok → Tashkent | 35~40일 | 20피트 단독 발차 가능, 위험품 가능 |
| **Sea+Rail+Truck** | Busan/Incheon → Qingdao → Kashgar → Andijan | 약 20일 | 리드타임 빠름, 남부 지역에 유리 |

---

## 2. TCR vs TSR 상세 비교 (운영팀 실무 기준)

### TCR
- 40피트 컨테이너 기준 운영
- **20피트 컨테이너는 pair로만 진행** (단독 발차 불가)
  - 중국에서 짝 컨테이너를 찾을 때까지 홀딩
- 중국 철도청 규제 엄격 → 위험품 또는 미허가 품목 진행 불가
- 편중 제한: 전후 5톤, 좌우 90mm 이내

### TSR
- 20피트 단독 발차 가능 (러시아-우즈벡 화물이 많아 짝 찾기 용이)
- 위험품 진행 가능 (사전 컨펌 필요)
- TCR 대비 수출 아이템 제재 덜 엄격
- BOLT SEAL (말뚝씰) 필수
- 영문 + 러시아어 CI/PL 필수

### Sea+Rail+Truck
- 운임은 높으나 리드타임 약 20일로 빠름
- 우즈베키스탄 남부 지역 (Andijan 등) 접근성 장점
- SINO, HL 에이전트 사용

---

## 3. 출발 전 필수 확인 (운영팀 실무 기준)

### TSR 진행 시
- [ ] 영문 아이템명 + 영문 HS코드 확인
- [ ] **packing당 1.5ton 미만** 확인
- [ ] 비위험물이라도 **permit 필요 여부** 확인
- [ ] 영문 + 러시아어 CI/PL 수령
- [ ] HS코드별 아이템명 정확히 기재 확인
- [ ] 컨테이너 번호 서류에 기재 확인

### TCR 진행 시
- [ ] 컨테이너 작업사진 확인
- [ ] 적재 시 전후좌우 무게 balance 확인
- [ ] 작업사진 필수 (컨테이너 번호 포함)
- [ ] 영문 + 러시아어 CI/PL (러시아어는 필수는 아니나 권장)
- [ ] ETSNG Code 확인 (우즈벡 행)

## 4. 자주 발생하는 문제 TOP 3 (운영팀 실무 기준)

1. **철송 Delay** — 국경 적체로 발차 지연 (가장 빈번)
2. **미수 (수금)** — 수하인 측 결제 지연
3. **TSR 컨테이너 상태 문제** — FESCO가 컨테이너 상태를 문제 삼아 추가 비용 요구

## 6. 통관 주의사항

- HS코드는 **우즈벡 기준으로 사전 검증** 필수 (한국/중국과 다를 수 있음)
- 신고 가액 낮으면 세관 의심 → 가격 입증 자료 준비
- 통관 기간: 일반 3~5일 / 검사 시 7~14일

---

## 7. 계절별/상황별 주의사항

- **겨울(12~2월)**: 국경 지연 빈번, 리드타임 +5~7일
- **나브루즈(3월 21일)**: 현지 업무 중단
- **국경 적체 심화 시**: 발차 지역(Qingdao, LYG)에서 발차 자체가 금지되어 추가 딜레이 발생
- **TSR 사용 시**: Vladivostok에서 wagon 부족 또는 물량 부족으로 발차 Delay 빈번

---

## 8. 추가 비용 발생 포인트

- Vladivostok 지연 시 지체료
- TSR 컨테이너 내부 packing이 러시아 철도청 규정 미충족 시 개봉 + 재작업 비용
- 우즈벡 국경 보관료 (지연 시)
- 통관 검사 비용
- CMR 재발행 비용 (오류 시)

---

## 9. 파트너 네트워크

- Tashkent 핸들링: MTL UZ 지점 (지점 직접 핸들링)
- 백업 파트너: 없음 (지점 직접 핸들링)
- 주말 연락: 주의 필요
