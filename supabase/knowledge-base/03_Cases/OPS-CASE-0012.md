# Case OPS-CASE-0012 — 삼성물산 타이어 스크래치 데미지 (내부 소통 문제)

## 1. Case Summary

삼성물산 납품 타이어에 스크래치 데미지 발생. MCI 내부 소통 미흡으로 (MCI본사-MCI카작-트럭 협력사 간) 데미지 촬영 미확보. 하역 동영상은 확보했으나 데미지 부분 미촬영. 화주(삼성물산 장병희 부장)가 추후 문제 제기 시 재논의하기로 함.

## 2. Cargo Information

- 화물: 타이어 (스크래치)
- Container: FBLU4215020
- BL: 0122 IB2MR

## 3. Problem

- 도착지 하역 시 타이어 스크래치 발견
- MCI 기사가 안전 문제로 데미지 부분 미촬영
- 하역 과정 동영상만 있음 (데미지 증빙 없음)

## 4. Root Cause

### Confirmed Cause
- MCI본사 - MCI카작 - 트럭협력사 간 소통 미흡
- 데미지 발생 시 촬영 절차 미준수

## 5. Actions Taken

| Step | Action | Result |
|---|---|---|
| 1 | 하역 동영상 확보 후 삼성물산에 전달 | 데미지 부분 미촬영 확인 |
| 2 | 화주가 일반 하역 동영상으로는 클레임 근거 부족 판단 | 현재 클레임 보류 |

## 6. Final Result

현재 클레임 보류. 수요가가 향후 문제 제기 시 재논의 예정.

## 7. Lesson Learned

1. 데미지 발견 시 **즉시 촬영** (안전 우선이나 기록 반드시 확보)
2. 데미지는 내부 보고 먼저 → 화주 통보 전 대응 방안 검토
3. MCI 협력 트럭사에 데미지 촬영 의무 교육 필요

## 8. New SOP Rule

> 도착지 데미지 발견 시:
> 1) 즉시 사진/동영상 촬영 (데미지 부위 클로즈업 포함)
> 2) 내부 보고 후 화주 통보 (순서 준수)
> 3) 데미지 촬영 없이 화주 통보 금지
